Getting started with HTML

HTMLとは

ウェブサイトの構成をブラウザーに伝えるために使うマークアップ言語

ブロック要素とインライン要素

pはブロックレベル要素になります

emの適当emはインラインです

属性

属性はコンテンツの中で表示させたくない、要素に関する追加情報を保有する。

ここをクリックしてmozillaのホームページに飛ばす

ここでhref、title、targetの属性を付けることでクリック時に別ページに飛ばしたり、 カーソルを合わせることでツールチップとして表示したり、 リンク先を新しいタブに飛ばしたり等々いろいろできる。

論理属性

属性値のない属性があり、それらを論理属性呼ばれている

disabled="disabled"
disabledのみ
inputオンリー

属性値を引用符で囲む

省略できる所もあるがミスを減らすためにも基本は書いた方がいい

また、単一引用符と二重引用符は混在させない限りどちらを使ってもかまわない

HTML文書の構成

html要素：ページ上の全ての内容を囲む要素。

head要素：閲覧者のための者ではなく、検索結果に現れるキーワードやページの説明文、 内容をスタイル設定するためのCSS、などが記載される。

meta要素：base, link, script, style, titleなど、他のHTMLメタ関連の要素では 表現する。

title要素：ページのタイトルを設定する。

body要素：テキスト、画像、動画、ゲーム、再生可能な音声トラックなどなど

HTML　特殊文字

< > " ' &

などなどHTMLの構文の一部で、それらをテキストとして記述する場合は アンパサンドで始まり、セミコロンで入力しなくてはいけない。
