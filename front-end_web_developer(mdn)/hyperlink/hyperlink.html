ハイパーリンク

リンクの解剖

ハイパーリンクによって、文書を他の文書やリソースにリンクしたり、 文書の特定の部分にリンクしたり、ウェブアドレスで利用できるようにする。

リンクの他にもtitle属性を持たせることでカーソルを合わせるだけでその先の情報を ツールチップとして表示できる。

URLとパス

文書フラグメント

HTML文書の上部だけでなく、特定の部分にリンクすることもできる。リンク先の要素に id属性を割り当てる。

Want to write us a letter? Use our mailing address.

絶対 URL vs 相対 URL

絶対URL : プロトコルとドメイン名を含む、ウェブ上の絶対位置で定義された位置を指す。 絶対 URL は、使用されている場所に関係なく、常に同じ場所を指します。

相対URLリンク元のファイルからの相対的な場所を指す。 相対 URL は内部で使用されているファイルの実際の場所によって、違う場所を指す。

リンクのいい習慣

明確なリンク語を使う
  1. ユーザーはリンクからリンクへと飛ぶ傾向にある。
  2. ユーザーは全ての文に目を通すわけではないので、リンクのような目立つモノに 説明的なリンクテキストは有効な手段になる。
  3. 検索エンジンはリンクテキストを使用して対象ファイルにインデックスを付ける。 その為、リンクテキストにキーワードを含める事で効果的な説明が期待できる。

例：

Firefox をダウンロード

こちらをクリック して Firefox をダウンロード

販売レポートをダウンロード (PDF, 10MB)

動画を見る（別なタブで HD 品質のストリーミングが開きます）

カーゲームで遊ぶ（Flash が必要）
